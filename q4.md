# Q4

| 步驟         | 三個人個別的金額 | 店家的金額 | 店員的金額 | 總計         |
| ------------ | ---------------- | ---------- | ---------- | ------------ |
| 一開始收錢   | 300              | 0          | 0          | 300\*3       |
| 付錢後       | 0                | 900        | 0          | 900          |
| 店員發現優惠 | 0                | 750        | 150        | 750+150      |
| 店員退錢後   | 30               | 750        | 60         | 30\*3+750+60 |

最後三人個別會得到 300-30(退還的錢)=270，所以總金額的算法會是 (250+30 退還)*3+60 才對，而不是 (250+20)*3+60
其中 30 元的差距就在此
